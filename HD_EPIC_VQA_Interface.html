HD-EPIC VQA Interface

This interface allows you to explore a sample of VQA questions. To start, select the VQA question JSON file .


One-click launcher: run START_HD_EPIC_INTERFACE.bat in this folder.

How It Works:

  • The Video Player (left) shows full videos.
  • The Video Clip Player (right) plays video clips with defined start/end times.
  • Timestamps appear below the Video Player. Click to pause at that frame.
  • Bounding Boxes are displayed if available.

Select an answer and click 'Next' to continue. Your accuracy will be tracked.
